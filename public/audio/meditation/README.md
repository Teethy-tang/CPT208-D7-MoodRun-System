# Meditation Audio

This folder contains ambient loop files for MoodRun's Meditation Mode.

## Files

- `rain.mp3`: rain ambience
- `ocean.mp3`: ocean ambience
- `forest.mp3`: forest ambience
- `wind.mp3`: wind ambience

## App Integration

The files are referenced in `src/features/meditation/ambientAudio.ts` and loaded from Vite's public asset base path.
